A Modified Shortest Path Problem

Preface

A constrained variant and reduction strategy back to classic shortest path.

Problem

Directed G=(V,E), w(e), c(e)∈{W,G,R}; alternate W→G→R; compute δ_BG(s,u).

Solution sketch

Transform to layered G'; run Bellman‑Ford; take min over target layers.

Theorem

Paths starting at a white node in G' correspond 1‑to‑1 to alternating paths in G.

Complexity

Same as Bellman‑Ford on G' (3n nodes, m edges): O(nm).
