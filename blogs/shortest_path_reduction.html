A Modified Shortest Path Problem

Preface

The shortest path problem is a cornerstone of graph theory. This post explores a constrained variant and a reduction strategy back to classic shortest path, with visuals.

Problem

Directed graph G = (V, E), edge weights w(e) and colors c(e) ∈ {W,G,R}. A path alternates W→G→R repeatedly. Compute δ_BG(s, u): the minimum‑weight alternating path from s to u, assuming no negative alternating cycles.

Solution sketch

Transform G to G' with three layers per vertex (White, Green, Red) and remap edges to preserve alternation. Then run Bellman‑Ford on G'.

Theorem

Paths starting at a white node in G' correspond 1‑to‑1 to alternating paths in G.

Complexity

Same as Bellman‑Ford on G' (3n nodes, m edges): O(nm).
